## Research Campaign Test Data

This folder contains 5 ready-to-use campaign fixture packs for the web app's "New Research Campaign" form.

Each campaign folder includes:
- `form-inputs.md`: Copy/paste values for title, description, question bank text, and timeline.
- `contacts.csv`: Upload this in the `Contact List (CSV)` field.

How to use in app:
1. Open `Dashboard -> Campaigns -> New Campaign`.
2. Use `Enter Text` mode for question bank.
3. Copy values from a campaign's `form-inputs.md`.
4. Upload that campaign's `contacts.csv`.
5. Submit, then continue into `Refine with AI`.
